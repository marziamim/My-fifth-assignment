Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model is a programming interface for web documents.It represents the page so that programs can change the document structure,style and content.The DOM repesents the document as nodes and objects that way

How do you select an element from the DOM ?

I can select an element from the DOM in few ways. Example:
1.getElementById()
2.getElementsByClassName()
3.getElementsByTagName()
4.querySelector()
5.querySelectorAll() etc.

.
What is event delegation in the context of the DOM, and why is it useful ?

Event delegation refers to the process of using event propagation(bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

How do you manipulate an elements attributes and style using the DOM ?

I can use getAttribute(), setAttribute(), removeAttribute() and 'style' property or adding/removing classlist etc. using techniques. I can dynamically update attributes and style in response to user actions or other event in the DOM.
